# talkingdata
TalkingData Analytics Cordova (PhoneGap) SDK  

版本号：V2.0.5  

发布日期：2015-10-15  

更新内容 ：  
1. 1. 优化灵动事件控件的识别、触发。  

注意：  
iOS需要添加CoreMotion.framework、QuartzCore.framework  

安装方法：  
cordova plugin add https://github.com/yforrest/talkingdata.git  

更新方法：  
cordova plugin remove TalkingData  
cordova plugin add https://github.com/yforrest/talkingdata.git
